Питание подсветки LCD

Dec 9th, 2011 10:27 pm | Comments

LCD-индикатор MI0350CT-3 имеет подсветку из соединенных последовательно светодиодов. Для питания такой подсветки необходим преобразователь напряжения с выходным напряжением от 16,8В до 21,6В и током до 20мА. Кроме этого, необходимо иметь возможность программного регулирования яркости подсветки.

На рисунке показана схема повышающего преобразователя, питающего нагрузку стабильным током 20мА. Преобразователь построен на основе микросхемы LM2733XMF.

Рис. 1. Принципиальная схема преобразователя (нажмите для увеличения).

Ток в нагрузке (в режиме стабилизации тока) задается резистором R4, делитель напряжения R2 R3 служит для ограничения выходного напряжения при разомкнутом выходе (режим стабилизации напряжения). Без цепи R2 R3 напряжение на выходе при отсутствии нагрузки будет неограниченно расти до выхода из строя микросхемы преобразователя.

Номиналы R1, L1, C1, C2, C3 выбраны в соответствии с Data sheet на LM 2733.

Номиналы резисторов R2, R3, R4 рассчитываются по достаточно простым формулам. Пусть выходное напряжение при отключенной нагрузке должно быт Vout=24 В. Тогда

R2 = R2 (Vout/Vfb - 1)

Напряжение Vfb = 1.23 В (см. data sheet). Выберем значение R2 = 12.1 кОм, получаем R2 = 220 кОм.

Значение R4 получаем, исходя из требуемого выходного тока.

Выходной ток Iout = 20 мА, R4 = (Vfb + Vf)/Iout = (1.23 + 0.2)/20e-3 = 71.5 Ом,

где Vf = 0.2 В - падение напряжения на диоде Шоттки (VD2).

Выбираем значение R4 = 72 Ом.

Таким образом, при сопротивлении нагрузки < R = (Vout - (R4 * I out + Vf))/Iout = (24 - (72 * 20e-3 + 0.2))/20e-3 = 1128 Ом преобразователь работает в режиме стабилизации тока, определяемого резистором R4, при большем сопротивлении нагрузки преобразователь переходит в редим стабилизации напряжения, определяемого делителем R2 R4.

Управление яркостью подсветки может осуществляться ШИМ-сигналом, подаваемым на вход /SHDN преобразователя.

Выходное напряжение при разомкнутом выходе равно 24.2В.

На рис. 2 приведен график зависимости КПД от выходной мощности.

Рис. 2. Зависимость КПД от выходной мощности.

Из таблицы 1 и рис. 2 видно, что КПД преобразователя не очень большой, порядка 60%. Согласно data sheet, максимум КПД приходится на выходной ток около 80мА и достигает почти 90%, при более низком токе выхода КПД резко падает.

При подключении цепи подсветки индикатора MI0350CT-3 ток составил 21мА, напряжение выхода 18,4В, что соответствует спецификациям индикатора.

По результатам испытаний можно сделать заключение, что ток в нагрузке поддерживается с достаточной точностью. К сожалению, очень маленький выходной ток не позволяет получить хороший КПД, но здесь, вероятно, ничего уже сделать нельзя.
