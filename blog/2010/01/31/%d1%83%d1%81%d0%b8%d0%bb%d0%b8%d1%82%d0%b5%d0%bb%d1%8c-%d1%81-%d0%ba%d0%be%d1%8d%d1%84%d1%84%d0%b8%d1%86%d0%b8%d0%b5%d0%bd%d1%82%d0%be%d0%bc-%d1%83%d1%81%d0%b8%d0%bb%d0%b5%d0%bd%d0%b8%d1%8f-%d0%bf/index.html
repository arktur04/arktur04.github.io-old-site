Усилитель с коэффициентом усиления, переключаемым с помощью микроконтроллера

Jan 31st, 2010 11:39 pm | Comments

Если в системе требуется программно переключить коэффициент усиления (Ку) усилителя, необязательно применять цифровой потенциометр. Если не требуется плавного изменения Ку, то достаточно и обычного операционного усилителя.
В статье показан способ реализации усилителя, коэффициент которого можно выбирать из нескольких значений с помощью микроконтроллера. В качестве примера применения такого усилителя можно привести узел аналогового ввода на базе встроенного в микроконтроллер АЦП, диапазон входных сигналов которого можно изменять программно, например, для подключения нескольких типов датчиков.

Порты ввода-вывода микроконтроллеров (GPIO), как правило, имеют следующую структуру (рис. 1).
 
Рис.1. Упрощенная структура линии порта микроконтроллера.
 
В зависимости от устанавливаемого программно состояния выхода замыкается либо ключ 1, либо ключ 2 (на самом деле, конечно, это полевые транзисторы). Если линия порта сконфигурирована как выход, и он находится в состоянии лог. 0, то замкнут ключ К2, если в состоянии лог. 1, то ключ К1, если линия порта сконфигурирована как входная, то оба ключа разомкнуты. Таким образом, программно переключая линию порта из состояния входа в состояние «выход, лог. 0», можно управлять ключом К2, оставляя К1 постоянно разомкнутым.

 

Рис. 2. Схема усилителя с выбором одного из двух значений Ку.

На рис.2. приведен пример схемы усилителя, у которого Ку можно изменять программно, выбирая одно из двух значений.

Когда линия порта сконфигурирована как вход, то есть находится в состоянии высокого импеданса,

Ky = (R2 + R3)/ R3;

Когда линия порта находится в состоянии лог. 0,

Ky = (R3 R2 + R1R2 + R1R3)/(R1R3);

Если R3 отсутствует, то:

если линия порта находится в состоянии высокого импеданса, то Ky = 1;

если линия порта находится в состоянии лог. 0, то

Ky = (R2 + R1)/ R1;

Можно получить больше градаций изменения Kу, если использовать несколько линий порта, как показано на рис. 3.

  

Рис. 3. Схема усилителя с выбором одного из множества значений Ку.

Заключение

В статье рассмотрена схемотехника и принцип действия, приведены расчетные формулы усилителя с коэффициентом усиления, который может выбираться из ряда значений с помощью микроконтроллера.

Литература

Хоровиц П., Хилл У. Искусство схемотехники: В 3-х томах: Т. 1. Пер. с англ.— 4-е изд., перераб. и доп.— М.: Мир, 1993.—413 с, ил. ISBN 5-03-002337-2
